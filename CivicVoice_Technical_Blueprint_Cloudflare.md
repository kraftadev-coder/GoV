Market Focus: Nigeria / Africa (Edge-First Performance)

1. Cloudflare-Native Architecture

By consolidating the stack on Cloudflare, we achieve the lowest possible latency in West Africa and eliminate egress costs for media proofs.

1.1 The "Edge-First" Stack

Hosting: Cloudflare Pages (Directly connected to your GitHub main branch).

Runtime: Cloudflare Workers (Edge Computing for the Amnesia Protocol).

Database: Cloudflare D1 (Serverless SQL - faster for reputation queries than NoSQL).

Object Storage: Cloudflare R2 (Storage for Video/Audio with $0 Egress Fees).

Security: Cloudflare WAF + Turnstile (Privacy-first CAPTCHA).

2. Frontend Layer (Next.js on Cloudflare Pages)

Next.js 14+ is fully supported on Cloudflare Pages using the @cloudflare/next-on-pages adapter.

2.1 PWA & Client-Side Edge

Edge Middleware: Geolocation is handled at the edge. The app knows if the user is in "Ikeja" or "Abuja" before the page even finishes loading.

Media Pipeline (On-Device):

Web Audio API: Client-side pitch shifting for "Sor Soke" mode.

FFmpeg.wasm: Client-side video transcoding to save user data.

3. Backend Logic (Cloudflare Workers)

Workers replace traditional backends. They are "Short-lived," meaning they don't hold data, which perfectly aligns with our Amnesia Protocol.

3.1 The "Amnesia" Endpoint (/api/report)

Ingress: Worker receives the multi-part form (Audio/Image + Geo-coords).

Validation: Worker checks the cf-ipcountry and cf-iplatitude/longitude headers (Cloudflare's built-in geo-data) to verify the "Witness" location.

Scrubbing: Worker streams the media to R2, stripping metadata on the fly using a lightweight WASM library.

Wipe:

Worker logs the post to D1.

The Worker execution finishes. Since Workers are stateless, the user's IP address and session data disappear from memory instantly. No persistent logs are created.

5. Cost Control & Solo Scaling (R2 Advantage)

Cloudflare R2 is the primary reason this is cost-effective for a solo builder:

S3/Firebase Storage: Charges you every time someone watches a video (Egress). A viral video in Nigeria could cost you $100s.

Cloudflare R2: Charges $0 for Egress. You only pay for the storage space used. 10GB of video evidence costs you $0.15/month.

6. Deployment & CI/CD

Local Dev: Use Wrangler (Cloudflare CLI) to simulate the edge locally.

GitHub Flow: * Develop on a feature branch.

Pull Request to main.

Cloudflare Pages automatically generates a Preview URL.

Merge to main deploys to production in < 60 seconds.

7. Security Hardening

Rate Limiting: Set via Cloudflare Dashboard to 5 posts per hour per IP to prevent spam during elections/protests.

Turnstile: Used on the "Submit" button. It verifies the user is human without storing cookies or tracking data, unlike Google reCAPTCHA.

Bot Management: Automatically challenges non-browser traffic (scripts/scrapers).

8. Summary for Developers

Building on Cloudflare allows the team to focus on the Witness UI. You don't need to configure servers, VPCs, or complex IAM roles. Every line of code in the GitHub repository is automatically "Edge-Optimized" for the African continent.
